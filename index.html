ipyaggrid

The power of ag-Grid in Jupyter notebooks

Get Started →

Accessible

Display pandas dataframes as dynamic HTML5 grids - Standard options are accessible through configuration.

Transparent

Access all ag-Grid features through its well crafted API without contraints - Full pass-through is implemented.

Customizable

Create your own Javascript functions to interact with the grid. Leverage examples in the ag-Grid docs.

# Install

# for notebook >= 5.3
$ pip install ipyaggrid

COMPATIBILITY NOTE

Tested on Chrome, Firefox, Opera, Min.
